# RealEstatePricingAnalysis
# Исследование динамики ценовых показателей жилой недвижимости (г. Рязань, вторичный рынок)

> **Учебная практика** — анализ данных, регрессионное моделирование и визуализация.
> 
> **Источник данных:** Выгрузка с ЦИАН (`offers123.xlsx`), 200 объектов.

---

## 📌 Описание проекта

Проект посвящен исследованию факторов, влияющих на стоимость квартир на вторичном рынке г. Рязани. На основе выгрузки с ЦИАН проводится:

- Очистка и предобработка данных (работа с пропусками, дубликатами, выбросами);
- Описательная статистика и визуализация распределений;
- Корреляционный анализ признаков;
- Построение и сравнение нескольких регрессионных моделей (линейных и логарифмических);
- Выбор наилучшей модели по скорректированному R² и оценка её качества.

Проект выполнен на Python с использованием `pandas`, `numpy`, `matplotlib`, `seaborn`, `scipy` и `statsmodels`.

---

## ⚠️ Важное замечание по данным

**Используется только файл `offers123.xlsx`.**  
CSV-версия (`offers123.csv`) **повреждена** из-за:
- неправильного разделителя (фактически `;`, а не `,`);
- переносов строк внутри текстовых полей, которые ломают структуру таблицы;
- преобразования номеров телефонов в экспоненциальный формат (`7.99E+10`).

В `.xlsx` все данные читаются корректно (телефоны остаются строками, переносы строк не нарушают структуру). Код рассчитан именно на этот формат.

📊 Результаты
1. Описательная статистика
Распределения цен, общей площади, количества комнат, этажности и других признаков.

Выявлены потенциальные выбросы по цене и площади.

2. Корреляционный анализ
Построены диаграммы рассеяния для 7 пар признаков.

Рассчитана матрица корреляций и тепловая карта.

Проверена статистическая значимость корреляций с ценой (критерий Пирсона, p-уровень).

3. Регрессионные модели
Сравнивались 5 моделей:

Модель	Переменные	R² (скорр.)
Базовая	total_area, floor	~0.54
Расширенная	+ rooms, total_floors, first_floor	~0.60
+ ремонт и балкон	+ is_brick, repair, has_balcony	~0.67
Логарифмическая	log_total_area, rooms	~0.65
Лог. расширенная + ремонт	log_total_area, rooms, total_floors, first_floor, repair, has_balcony	~0.75
Наилучшая модель — логарифмическая расширенная с ремонтом и балконом (Adj. R² ≈ 0.75).

4. Оценка качества прогноза (наилучшая модель)
RMSE: ~1 200 000 руб.

MAE: ~850 000 руб.

MAPE: ~12–15% (в зависимости от данных)

🧠 Выводы
Основной фактор, определяющий цену, — общая площадь (эластичность ≈ 0.7–0.8 в логарифмической модели).

Ремонт и наличие балкона дают значимую положительную надбавку к цене.

Первый этаж снижает стоимость (отрицательный коэффициент).

Этажность дома и количество комнат также влияют, но в меньшей степени.

Логарифмическая трансформация улучшает качество модели и делает интерпретацию коэффициентов более естественной (эластичности).
